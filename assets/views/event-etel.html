ArdioCrats

Round 1: - Preliminary quiz 

The first round of the event is based on general quiz based on Basic Of electronics. Like analog, digital etc. The quiz contains 50 MCQ and 10 general one statement questions. For that we have to give a time limit of 45min.


Round 2: - Arduino Coding

A 10-12 teams are shortlisted for this round. A problem statement would be asked on basic Arduino programming (like LED blink, display, serial communication etc.). They have to display the working Arduino code in Arduino IDE. It runs for a maximum of 1 hr.

 
Round 3: - Arduino Implementation

In final round problem statement would be given and they have to write code and implement circuit on Proteus tool. Round 3 is almost on the base of practical application like LCD display, motor controlling, led controlling etc. So, the problem statement will be given on the spot and the participants will have to solve the problem(s) within the specified time limit. A 6-8 teams are shortlisted for this round. A problem statement would be given to teams. They have to solve the problem and write the code for it on Arduino IDE. After that they have to implement circuit on PROTEUS..
